    1. [h]yper[t]ext [m]arkup [l]anguage
    2. [h]yper[t]ext : 시공간을 초월하는 텍스트 ( 클릭하면 완전히 다른 곳으로 넘어가는 (링크) 같은 것 )
    3. [m]arkup: 태그 형식의
    4. [l]anguage : 언어
    5. 웹페이지를 구조를 위한 표준 마크업(태그) 언어
    6. 시작하는 태그와 종료(닫기)태그가 있다. (시작태그) 내용content-태그도 들어감 (/닫기태그)
    7. 시작태그 + 내용물 + 닫기태그 = 요소(element)
