js 코드는 <script> </script> 태그 안에 코딩한다. <script type="text/javascript"></script> 였던 적도 있음 (지금은 필수 X) <body></body> , <head></head> 태그 안에 추가하는 것이 국룰 <script></script> 요소를 </body>위에 배치하면 스크립트 해석으로 인해 표시 속도가 향상된다. 여러 웹 페이지에서 공통ㄷ적으로 사용되는 js코딩이 있다면 참조 가능. html, js코딩 분리되니 가독성 올라감 (유지보수 유리) 메모리에 js파일 하나 올려놓는 것이기 때문에 html파일 마다 매번 로딩되지 않아 유리함
